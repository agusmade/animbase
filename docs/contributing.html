Getting Started

  1. Fork this repository
  2. Run yarn install
  3. Use yarn dev to start the development server
  4. Submit your changes in a new branch via a pull request

Code Standards

  • Use prettier (already integrated via .editorconfig and .prettierrc)
  • Run linter: yarn lint
  • Run tests: yarn test or yarn test:cov
  • Check coverage: yarn test:cov

Suggestions

  • Fix or open an issue for bugs
  • Add tests where applicable
  • Follow the existing code style and structure
  • Keep pull requests focused and concise
  • Document any new public APIs
